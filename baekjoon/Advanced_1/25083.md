문제
=====
아래 예제와 같이 새싹을 출력하시오.

입력
=====
입력은 없다.

출력
=====
새싹을 출력한다.

예제 입력 1 
-------

예제 출력 1 
------
```
         ,r'"7
r`-_   ,'  ,/
 \. ". L_r'
   `~\/
      |
      |
```

내가 제출한 답
--------------
```cpp
#include <iostream>

int main()
{
	std::cout << "         ,r'\"7" << std::endl <<
		"r`-_   ,'  ,/" << std::endl <<
		" \\. \". L_r'" << std::endl <<
		"   `~\\/" << std::endl <<
		"      |" << std::endl <<
		"      |";
}
```
