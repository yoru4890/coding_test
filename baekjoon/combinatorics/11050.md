문제
============
자연수 
$N$과 정수 
$K$가 주어졌을 때 이항 계수 
$\binom{N}{K}$를 구하는 프로그램을 작성하시오.

입력
==========
첫째 줄에 
$N$과 
$K$가 주어진다. (1 ≤ 
$N$ ≤ 10, 0 ≤ 
$K$ ≤ 
\(N\))

출력
========
 
$\\binom{N}{K}\$를 출력한다.

예제 입력 1 
-----------
```
5 2
```
예제 출력 1 
----------
```
10
```

내가 제출한 답
--------------
```cpp
#include <iostream>

using namespace std;

int Factorial(int N);


int main()
{
	int N{}, M{}, count{};

	cin >> N >> M;

	count = Factorial(N) / Factorial(M) / Factorial(N - M);

	cout << count;
}

int Factorial(int N)
{
	if (N == 0)
	{
		return 1;
	}

	return N * Factorial(N - 1);
}
```
