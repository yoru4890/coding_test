문제
===========
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

입력
==========
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

출력
===========
첫째 줄에 N!을 출력한다.

예제 입력 1 
----------
```
10
```
예제 출력 1 
--------
```
3628800
```
예제 입력 2 
-----------
```
0
```
예제 출력 2 
----------
```
1
```

내가 제출한 답
---------------
```cpp
#include <iostream>

using namespace std;

unsigned long long int Factorial(int num);

int main()
{
    ios::sync_with_stdio(false);

    int N{};

    cin >> N;

    cout <<Factorial(N);
}

unsigned long long int Factorial(int num)
{
    if (num == 0)
    {
        return 1;
    }

    return num*Factorial(num-1);
}
```
