문제
========
3×N 크기의 벽을 2×1, 1×2 크기의 타일로 채우는 경우의 수를 구해보자.

입력
=========
첫째 줄에 N(1 ≤ N ≤ 30)이 주어진다.

출력
========
첫째 줄에 경우의 수를 출력한다.

힌트
==========
아래 그림은 3×12 벽을 타일로 채운 예시이다.

![pic](https://onlinejudgeimages.s3-ap-northeast-1.amazonaws.com/upload/images/2663_1.jpg)

예제 입력 1 
----------
```
2
```
예제 출력 1 
--------
```
3
```
