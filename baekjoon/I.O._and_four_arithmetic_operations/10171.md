문제
======
아래 예제와 같이 고양이를 출력하시오.

입력
=======
없음.

출력
=======
고양이를 출력한다.

예제 입력 1 
-------

예제 출력 1 
--------

```
\    /\   
 )  ( ')   
(  /  )   
 \(__)|
```

내가 제출한 답
--------
```cpp
#include <iostream>

int main()
{
	std::cout << "\\    /\\" << std::endl <<
		" )  ( ')" << std::endl <<
		"(  /  )" << std::endl <<
		" \\(__)|";

}
```
