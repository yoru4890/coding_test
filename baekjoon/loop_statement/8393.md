문제
========
n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

입력
=========
첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.

출력
==============
1부터 n까지 합을 출력한다.

예제 입력 1 
-------------

3

예제 출력 1 
------------

6

내가 제출한 답
------------

```cpp
#include <iostream>

int main()
{
	int num{};
	int sum{};

	std::cin >> num;

	if (num < 1 || num>10000)
	{
		return 0;
	}

	for (int i = 1; i <= num; i++)
	{
		sum += i;
	}

	std::cout << sum;
}
```
